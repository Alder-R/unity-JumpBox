# Jump Box
 유니티3D 프로젝트 - 점프 박스<br><br>


기획:
- 유니티 3D를 활용해 3인칭(Back view) 플랫포머 게임을 제작, 이를 통해 기본적인 기능 활용 및 학습
- 플레이어는 공을 WASD 또는 방향 키로 이동할 수 있으며, SPACE키로 점프가 가능하다.
- 맵에 흩어져 있는 코인을 모아 상점을 이용할 수 있다.
- 맵에서 떨어지면 체크포인트 부분으로 순간이동한다.
